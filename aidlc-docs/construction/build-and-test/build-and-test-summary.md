## 次のステップ

1. `pip install -e ".[dev]"` で依存関係をインストールする
2. `pytest tests/ -v` で全ユニットテストを実行し、45件全てパスすることを確認する
3. `python main.py` でアプリを起動し、統合テストのシナリオを手動で確認する
4. 問題があれば対応するソースコードを修正する
